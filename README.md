# LIGHTSPEED HACKATHON - SoYouThinkYouCanML  
Demo datasets for LightSpeed Hackathon Challenge *So You Think You Can ML*

## List of datasets and sources

- **adult_income.csv**: https://www.kaggle.com/datasets/uciml/adult-census-income (CC BY 4.0)  
  - Source: U.S. Census Bureau data (1994 Census database).  
  - Hosted by: UCI Machine Learning Repository & Kaggle.  
  - Format: ~48,842 instances, 14 attributes (demographic and employment-related).  
  - TARGET: `income` (binary classification: `<=50K` vs `>50K`).  

- **credit_default_uci.csv**: https://archive.ics.uci.edu/ml/datasets/default+of+credit+card+clients (CC BY 4.0)  
  - Source: Taiwan (2005) credit card default study.  
  - Hosted by: UCI Machine Learning Repository.  
  - Format: ~30,000 instances, 23 attributes (demographics, credit history, repayment, bill statements, payments).  
  - TARGET: `default.payment.next.month` (binary classification: default vs no default).  

- **bank_marketing.csv**: https://archive.ics.uci.edu/ml/datasets/bank+marketing (CC BY 4.0)  
  - Source: Portuguese banking institution’s direct marketing campaigns (2008–2013).  
  - Hosted by: UCI Machine Learning Repository.  
  - Format: ~45,211 instances, 17 attributes (socioeconomic and marketing interaction features).  
  - TARGET: `y` (binary classification: client su  bscribed to a term deposit `yes`/`no`).  

- **census_income.csv**: https://archive.ics.uci.edu/dataset/117/census+income+kdd (CC BY 4.0)  
  - Source: U.S. Census Bureau’s Current Population Survey (1994–1995).  
  - Hosted by: UCI Machine Learning Repository.  
  - Format: ~299,285 instances, 42 attributes (expanded demographic, employment, and weighting variables).  
  - TARGET: `label` (binary classification: `<=50K` vs `>50K`).  

- **diabetes_health_indicators.csv**: https://archive.ics.uci.edu/dataset/891/cdc+diabetes+health+indicators  
  - Source: U.S. Centers for Disease Control and Prevention (CDC), via the Behavioral Risk Factor Surveillance System (BRFSS), 2015.  
  - Hosted by: UCI Machine Learning Repository ("CDC Diabetes Health Indicators") and mirrored on Kaggle.  
  - Format: ~253,680 survey responses with 21 demographic and health-related features like high blood pressure, cholesterol check, BMI, smoking, income, education, age, sex, among others.  
  - TARGET: `Diabetes_012` indicating:  
    - 0 = no diabetes or gestational diabetes  
    - 1 = prediabetes  
    - 2 = diabetes
- **covtype.csv**: https://www.kaggle.com/datasets/uciml/forest-cover-type-dataset (CC BY 4.0)  
  - Source: U.S. Forest Service Region 2 Resource Information System.  
  - Hosted by: UCI Machine Learning Repository & Kaggle.  
  - Format: ~581,012 instances, 54 attributes (cartographic variables).  
  - TARGET: `label` (multi-class classification, 7 forest cover types).
